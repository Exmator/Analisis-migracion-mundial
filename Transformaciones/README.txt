Comentarios generales del archivo "Codigo_Transformación.R"

- Dentro de este código se va a encontrar una serie de comandos usados para la transformación de cada una de las bases del proyecto y la construcción de la base final.

- El código esta seccionado en base a la(s) tabla(s) que se van a construir para representar cada uno de los criterios del proyecto. Por ejemplo, debajo del comentario ---Stock migratorio--- se encontrara todo el código relacionado a la construcción del apartado relacionado al stock migratorio en el proyecto

- Cada bloque de código contiene comentarios que explican de manera general que realiza cada uno de estos.

- Las bases importadas no son necesariamente iguales a las ofrecidas por la ONU, esto se debe a que ya pasaron por un proceso de limpieza manual previo que es explicado en el documento del proyecto. Se adjuntaran los archivos usados en este código de R dentro de una carpeta en el anexo del documento llamada "archivos_codigo_transformacion"

- Para el correcto funcionamiento del código es necesario poner los archivos dentro de la carpeta "archivos_codigo_transformacion" dentro de la carpeta por defecto de R en us computador, o agregar la ruta completa de donde esta localizado cada uno de los archivos.
